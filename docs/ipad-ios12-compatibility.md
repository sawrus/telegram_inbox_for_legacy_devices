# Совместимость с iPad Safari iOS 12

Фронтенд сделан максимально простым:

- серверный HTML через Jinja2;
- без SPA-фреймворков;
- без WebSocket;
- без обязательного современного JavaScript;
- автообновление через `<meta http-equiv="refresh">`;
- CSS использует базовые свойства, поддерживаемые Safari iOS 12.

Основной экран использует две панели на ширине iPad. На узких экранах панели складываются вертикально. Поиск чатов
работает через обычные GET-формы; небольшой JavaScript только отправляет форму после ввода и не обязателен для работы.

Если страница не обновляется, проверьте значение `REFRESH_SECONDS` и откройте основной экран заново.
