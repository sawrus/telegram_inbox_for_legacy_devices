# Два чата на одном экране

## Цель

Пользователь может открыть основной экран и одновременно читать два Telegram-чата: один в левой панели, другой в
правой. Это помогает сравнивать или мониторить переписки на старых устройствах без SPA и WebSocket.

## Acceptance Criteria

- Основной маршрут `/` показывает левую и правую панели чатов с толстой вертикальной разделительной линией.
- Каждая панель имеет собственный поиск по названию чата и собственный выбор чата.
- Поиск в одной панели не меняет выбранный чат и результаты другой панели.
- Стартовые чаты берутся из `LEFT_CHAT_ID` и `RIGHT_CHAT_ID`, если URL не задаёт `left_chat_id` и `right_chat_id`.
- Старый маршрут `/chats/<id>` перенаправляет на основной экран с выбранным левым чатом.
- Сообщения показывают входящие и исходящие записи.
- Исходящие сообщения отмечены как `Моё`, имеют светло-серый фон и сдвиг вправо на `20px`.
- Время сообщений отображается в часовом поясе `MESSAGE_TIMEZONE`, по умолчанию `Europe/Samara`.
- Страница сохраняет выбранные чаты и поисковые запросы при автообновлении через query string.
- На узких экранах панели складываются вертикально.

## Non-Goals

- Отправка сообщений из интерфейса.
- Редактирование, удаление, пересылка, реакции.
- WebSocket или SPA.
- Больше двух панелей.
- Поиск по тексту сообщений.
